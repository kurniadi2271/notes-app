# Aplikasi Catatan (Notes App)

Proyek ini adalah sebuah aplikasi pencatatan sederhana (Notes App) yang dibangun sebagai bagian dari pembelajaran fundamental pengembangan web front-end. Aplikasi ini memungkinkan pengguna untuk membuat, melihat, mencari, mengarsipkan, dan menghapus catatan.

## âœ¨ Fitur

- **Buat Catatan Baru**: Menambahkan judul dan isi catatan.
- **Tampilkan Catatan**: Menampilkan semua catatan aktif dan yang diarsipkan dalam daftar terpisah.
- **Pencarian Catatan**: Mencari catatan berdasarkan judul secara *real-time*.
- **Arsipkan Catatan**: Memindahkan catatan dari daftar aktif ke arsip.
- **Hapus Catatan**: Menghapus catatan secara permanen.
- **Penyimpanan Lokal**: Semua catatan disimpan di `localStorage` browser, sehingga data tidak akan hilang saat halaman dimuat ulang.
- **Desain Responsif**: Tampilan yang dapat menyesuaikan diri dengan berbagai ukuran layar.
